# GCT-FKP
We propose a graph convolution transformer to solve the forward kinematics problem of Gough-Stewart platform.
# Dependencies
* CUDA 11.3

* Python 3.9.13 

* Pytorch 1.11.0

* Numpy 1.22.4
# Dataset
Please click and download our [dataset](https://www.kaggle.com/datasets/huizhizhu/gsp-fkdataset).
